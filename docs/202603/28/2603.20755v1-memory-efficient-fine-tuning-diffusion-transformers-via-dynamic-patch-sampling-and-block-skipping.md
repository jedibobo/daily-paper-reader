## 摘要
Diffusion Transformers (DiTs) 显著提升了文本生成图像 (T2I) 的质量，实现了高质量的个性化内容创作。然而，微调这些模型需要极高的计算复杂度和内存，限制了其在资源受限环境下的实际部署。为了应对这些挑战，我们提出了一种名为 DiT-BlockSkip 的内存高效微调框架，该框架集成了时间步感知的动态 Patch 采样，并通过预计算残差特征实现了 Block 跳过。我们的动态 Patch 采样策略根据扩散时间步调整 Patch 大小，然后将裁剪后的 Patch 调整为固定的低分辨率。这种方法减少了前向和后向传播的内存占用，同时允许模型在较高时间步捕捉全局结构，在较低时间步捕捉细粒度细节。Block 跳过机制有选择地微调关键的 Transformer Block，并为跳过的 Block 预计算残差特征，从而显著降低了训练内存。为了识别用于个性化的关键 Block，我们引入了一种基于交叉注意力掩码 (cross-attention masking) 的 Block 选择策略。评估结果表明，我们的方法在定性和定量上均实现了具有竞争力的个性化性能，同时大幅减少了内存使用，推动了大规模 Diffusion Transformers 在端侧设备（如智能手机、物联网设备）上的可行性。

## Abstract
Diffusion Transformers (DiTs) have significantly enhanced text-to-image (T2I) generation quality, enabling high-quality personalized content creation. However, fine-tuning these models requires substantial computational complexity and memory, limiting practical deployment under resource constraints. To tackle these challenges, we propose a memory-efficient fine-tuning framework called DiT-BlockSkip, integrating timestep-aware dynamic patch sampling and block skipping by precomputing residual features. Our dynamic patch sampling strategy adjusts patch sizes based on the diffusion timestep, then resizes the cropped patches to a fixed lower resolution. This approach reduces forward & backward memory usage while allowing the model to capture global structures at higher timesteps and fine-grained details at lower timesteps. The block skipping mechanism selectively fine-tunes essential transformer blocks and precomputes residual features for the skipped blocks, significantly reducing training memory. To identify vital blocks for personalization, we introduce a block selection strategy based on cross-attention masking. Evaluations demonstrate that our approach achieves competitive personalization performance qualitatively and quantitatively, while reducing memory usage substantially, moving toward on-device feasibility (e.g., smartphones, IoT devices) for large-scale diffusion transformers.

随着 Diffusion Transformers (DiT)（如 FLUX, SANA）在图像生成领域的兴起，其强大的表达能力带来了极高的计算和显存开销。特别是在进行**个性化微调**（如 DreamBooth）时，即便使用 LoRA 等参数高效微调技术，由于需要完整的前向和后向传播，显存占用依然超出了大多数消费级 GPU 和移动端设备的承受范围。本文旨在解决如何在保持生成质量的前提下，大幅降低 DiT 模型微调时的显存占用，推动大模型在端侧设备上的可行性。

### 2. 论文提出的方法论：DiT-BlockSkip
论文提出了一个名为 **DiT-BlockSkip** 的框架，包含两个核心组件：

*   **动态 Patch 采样 (Dynamic Patch Sampling)：**
    *   **核心思想：** 扩散模型在不同时间步（$t$）学习的内容不同（高 $t$ 学全局结构，低 $t$ 学局部细节）。
    *   **技术细节：** 根据时间步 $t$ 动态调整裁剪 Patch 的大小。公式定义为：$f(s_{min}, s_{max}, t) = s_{min} + \frac{t}{T} \cdot (s_{max} - s_{min})$。
    *   **流程：** 在高 $t$ 时采样大尺寸 Patch 以捕捉结构，在低 $t$ 时采样小尺寸 Patch 以捕捉细节。所有采样后的 Patch 均被缩放（Resize）至统一的低分辨率（如 256x256）进行训练，从而显著减少激活值显存。

*   **Block 跳过与残差特征预计算 (Block Skipping with Residual Feature)：**
    *   **核心思想：** 并非所有 Transformer Block 对个性化任务都同样重要。
    *   **关键技术：** 
        1.  **Block 选择策略：** 通过“交叉注意力掩码”实验发现，DiT 的**中间层**对个性化主题的表达至关重要，而开头和结尾的层相对冗余。
        2.  **残差预计算：** 对于选定跳过的 Block，预先计算其输入与输出之间的差值（残差 $\Delta f$）。在微调阶段，直接将该残差加到输入特征上，从而无需加载这些 Block 的参数，也无需进行前向和后向计算。
    *   **流程：** 仅对未跳过的关键 Block 注入 LoRA 并更新参数。
