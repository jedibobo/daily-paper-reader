## 摘要
轻量级神经网络加速器对于资源受限且有功耗约束的边缘设备至关重要。虽然量化和二值化可以有效降低硬件成本，但它们仍然依赖于传统的人工神经网络（ANN）计算模式。最近提出的 Kolmogorov-Arnold 网络（KAN）展示了一种基于可学习非线性函数的新型网络范式。然而，其硬件部署的计算开销较大。受 KAN 启发，我们提出了 BiKA，这是一种无乘法架构，它使用二值化的可学习阈值替代非线性函数，引入了一种仅需比较器和累加器的极轻量级计算模式。我们在 Ultra96-V2 上的 FPGA 原型表明，与二值化和量化神经网络脉动阵列加速器相比，BiKA 在保持具有竞争力的准确率的同时，分别减少了 27.73% 和 51.54% 的硬件资源消耗。BiKA 为边缘设备上硬件友好的神经网络设计提供了一个极具前景的方向。

## Abstract
Lightweight neural network accelerators are essential for edge devices with limited resources and power constraints. While quantization and binarization can efficiently reduce hardware cost, they still rely on the conventional Artificial Neural Network (ANN) computation pattern. The recently proposed Kolmogorov-Arnold Network (KAN) presents a novel network paradigm built on learnable nonlinear functions. However, it is computationally expensive for hardware deployment. Inspired by KAN, we propose BiKA, a multiply-free architecture that replaces nonlinear functions with binary, learnable thresholds, introducing an extremely lightweight computational pattern that requires only comparators and accumulators. Our FPGA prototype on Ultra96-V2 shows that BiKA reduces hardware resource usage by 27.73% and 51.54% compared with binarized and quantized neural network systolic array accelerators, while maintaining competitive accuracy. BiKA provides a promising direction for hardware-friendly neural network design on edge devices.

    *   传统的 **ANN/MLP** 依赖乘累加（MAC）运算，硬件开销大。
    *   新兴的 **KAN（Kolmogorov-Arnold Network）** 使用可学习的非线性函数代替固定激活函数，虽然参数效率高，但在硬件上实现非线性函数极其耗费资源（如需大量 LUT）。

### 2. 方法论
*   **核心思想**：提出 **BiKA（Binarized KAN）**。将 KAN 中昂贵的可学习非线性函数简化为**二进制可学习阈值（Binary Learnable Thresholds）**。
*   **关键技术细节**：
    *   **数学近似**：论文证明了任何连续非线性函数都可以通过一组加权阈值函数的总和来近似。
    *   **架构简化**：通过将权重近似为整数并进一步简化，BiKA 将神经元的计算模式从“乘法+激活”转变为“**比较+累加（Compare-Accumulate, CAC）**”。
    *   **计算公式**：BiKA 神经元的输出表示为 $Out = \sum Sign(W_i A_i + b_i)$，其中 $W_i$ 和 $b_i$ 共同决定了可学习的阈值。
    *   **训练技术**：使用 PyTorch 和 CUDA 开发了自定义层（BiKALinear/BiKAConv2d）。由于 $Sign$ 函数不可导，采用了**直通估计器（STE）**，利用 hard-tanh 的导数进行反向传播。
    *   **硬件实现**：设计了基于**脉动阵列（Systolic Array）**的加速器，处理单元（PE）仅包含比较器和累加器，完全去除了乘法器、XNOR 单元或复杂的 LUT 查表。

*   **充分性评价**：实验设计较为全面，不仅覆盖了简单的 MNIST，还挑战了较复杂的 CIFAR-10，并从算法准确率和硬件物理指标（面积、功耗、延迟）两个维度进行了客观对比。

### 6. 主要结论与发现
*   **硬件优势**：BiKA 加速器比 BNN 减少了 **27.73%** 的 LUT 资源，比 8-bit QNN 减少了 **51.54%**。
*   **能效表现**：BiKA 具有最低的面积延迟积（ADP）和功耗延迟积（PDP），显示出极高的能效比。
*   **准确率表现**：
    *   在 MNIST 上，BiKA 表现优异，与 BNN 的差距小于 1%。

*   **推理速度**：BiKA 比 8-bit QNN 快 2.17x-3.30x，但由于 BNN 可以利用 SIMD/Popcount 并行加速，BiKA 的绝对速度慢于优化后的 BNN。

### 7. 优点
*   **范式创新**：成功将 KAN 的数学思想转化为极简的硬件原语（比较器），打破了传统 MAC 计算的局限。
*   **极低成本**：为资源极度匮乏的边缘芯片提供了一种可行的方案，LUT 占用极低。
*   **数学支撑**：提供了从非线性函数到阈值函数转化的数学推导，增强了方法的可信度。

### 8. 不足与局限
*   **复杂任务表现不佳**：在 CIFAR-10 等复杂数据集上准确率下降明显，说明该架构在处理高维特征时表达能力受限。
*   **训练敏感性**：BiKA 对学习率和 Batch Size 非常敏感，训练调优难度可能高于传统网络。
*   **推理延迟**：虽然资源省，但在相同并行度下，其推理延迟并未超越高度优化的 BNN。
*   **应用限制**：目前主要验证了小型 MLP 和简单 CNN，尚未在大规模视觉模型（如 ResNet）或 Transformer 上验证。
